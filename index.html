ju1ius' CSS Parser

A CSS parser and library for PHP

Download this project as a .zip file Download this project as a tar.gz file

Install

The easiest way is via Composer: 

$ git clone git://github.com/ju1ius/css-parser.git
$ cd css-parser
$ composer.phar install

Or you can clone the two projects separately:

$ git clone git://github.com/ju1ius/css-parser.git
$ git clone git://github.com/ju1ius/libphp.git ju1ius-libphp

Setup

This library is PSR-0 compatible. If you installed via Composer, you just need to include: 

$autoloader = require '/path/to/css-parser/lib/vendor/.composer/autoload.php';

Otherwise you'll need a PSR-0 compatible autoloder, like the Symfony/CladdLoader component: 

require_once '/path/to/Symfony/Component/ClassLoader/UniversalClassLoader.php';
$loader = new Symfony\Component\ClassLoader\UniversalClassLoader();
$loader->registerNamespace('ju1ius', array(
'/path/to/css-parser/lib',
'/path/to/ju1ius-libphp/lib',
));
$loader->register();
