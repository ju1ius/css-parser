Safely cleans up properties before creating shorthand properties.

_safeCleanup(Array $aProperties, $sShorthand) : bool

Avoids creating a shorthand if:

  • More than one shorthand is already present, as it is generally done on purpose, ie for vendor specific values. 
          background: -webkit-linear-gradient(#000, #fff);
      background: -moz-linear-gradient(#000, #fff);
  • More than one identical properties are found, for the very same reasons
